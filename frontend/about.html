PhishGuard

AI-Powered Phishing Detection & Protection

Creator & Developer
👨‍💻
Anagh Khandelwal
Full-Stack Developer & Cyber Security Researcher
Designed and built PhishGuard — an AI-powered phishing detection system combining machine learning, heuristic analysis, and Gemini AI to protect users from cyber threats in real time.
Features
  • Real-time URL phishing detection using ML + 20+ heuristic signals
  • Email threat analysis with link extraction and scanning
  • PDF document analysis powered by Gemini AI vision
  • Deep Scan: WHOIS domain age lookup + redirect chain tracing
  • Image analysis for visual phishing detection
  • Chrome extension with auto-scan, blocking, and badge alerts
  • Gmail integration with inline threat analysis
  • Phishing site blocker — redirects dangerous sites before they load
  • Real-time statistics dashboard with server uptime and threat counts
